What you can expect from the new site

#General
#Lecture page
#Quiz page

  • Changing the language from xml to html5

    Simplfies the code and makes it more semantic. Creates consistency to make it easier for designers to update the pages. Easier to update the contents of the page.

  • Improved SEO

    Improving the structure of the page to be more semantic for the search engine bots to discover the website. Added in a meta tag with a description of the course website as well as a descriptive title.

  • Nice print style

    — Black and white.
    — No footer, nav, header.
    — Page breaks are avoided after headers.
    — Hyperlinks has the URL.

  • Responsive design

    — Compatible with mobile devices
    : collapsed navigation menu
    : No footer or header
    : images are resized to fit mobile view

  • Cache control

    Remember the login information and faster for the users who revisit the site

  • Use SASS for styling

    Using mixins and variables. Easier to maintain and more powerful css.

  • Warm colour pallete

    Studies show that warm colours improve memory (The Influence of Colour on Memory Performance https://www.ncbi.nlm.nih.gov/pmc/articles/PMC3743993/#b21-mjms-20-2-003)

  • Use PHP includes

    Easier to update specific code sections for many pages at once

  • Sitemap in footer

    Improves SEO - a direct link to each lecture is easy for students to go where they want to see and identifies which pages are important to the site.

  • Search button (to be implemented)

    Allow the students to quicky find which lecture and each topic is covered.

  • Notes for each lecture section (to be implemented)

    Allow the sturents to write the notes specific to each section. This keeps notes well organized for better learning and referencing during study.

  • Print lecture button on top

    One click action to print the entire lecture.

  • Accordion menu on lecture pages

    Improve the readability of the page for mobile devices.

  • Previous Quizzes and Grades

    Allow the students easy access to past quizzes in order to study and learn from their mistakes.

  • Save the answer button

    The answers are not lost once saved

  • Error page

    Site specific error page has similar styling to the rest of the site and can redirect user back to the homepage.

  • Class selection page

    Opens up first in order to ensure that the student is looking at the correct notes for the class they belong to.
